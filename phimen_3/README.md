# Phitron Mentorship Problem Solving 3 Solutions

This repository contains C++ solutions to various competitive programming problems. Each file corresponds to a specific problem statement.

## Problem List

| Problem Name | File Name | Description |
|--------------|-----------|-------------|
| **A/B Problem** | `A_B.cpp` | Calculates integer division, remainder, and floating-point division of two integers. |
| **Bone Appetit** | `bone_appetit.cpp` | Calculates the total number of treats collected from two sectors with different treat counts. |
| **Food Costs** | `food_costs.cpp` | Calculates the weekly food cost based on mess fees (6 days) and restaurant splurges (1 day). |
| **Air Quality Index** | `air_quality_index.cpp` | Checks if the Air Quality Index (AQI) is strictly below a government-mandated limit. |
| **All New CodeChef** | `all_new_codechef.cpp` | Compares the runtime of code on old and new judging systems to determine which is faster. |
| **Approximate Answer** | `approximate_answer.cpp` | Determines if an obtained answer is within a specified absolute difference from the correct answer. |
| **Ageing** | `ageing.cpp` | Calculates Chefina's age when Chef reaches a certain age, given a fixed age difference. |
| **Chess Pairing** | `chess_pairing.cpp` | Minimizes the number of rated players playing against other rated players in a tournament. |
| **Dice Number** | `dice_number.cpp` | Determines the winner of a dice game based on the largest number formed from 3 rolls. |
| **Fair Share Settlement** | `fair_share_settlement.cpp` | Calculates the net amount paid by a host after friends reimburse their rounded-down shares. |

## How to Run

To compile and run any of the solutions, you need a C++ compiler (like `g++`).

### Compilation
```bash
g++ -o solution <filename>.cpp
```
For example:
```bash
g++ -o A_B A_B.cpp
```

### Execution
```bash
./solution
```
(On Windows, use `solution.exe`)

You can provide input via standard input (keyboard) or redirect from a file:
```bash
./solution < input.txt
```

## Requirements
- C++ Compiler (GCC, Clang, MSVC)
- Standard C++ Library
